<Note>

  **+ Nova campanha** e inserir um nome para a campanha.
</Note>

## Visualizando o menu de nova campanha

Ao inseir o nome da campanha o usuário deverá pressionar o botão **Continuar** para ser direcionado para o menu onde poderá montar um campanha. Realizando esta ação o usuario visualizará os campos:

<Frame>
  <img
    className='block dark:hidden'
    src='images/campain/campain-menu-top-light.png'
    alt='Campain menu top light'
  />
  <img
    className='hidden dark:block'

    alt='Campain menu top dark'
  />
</Frame>

No topo do menu será mostrado o nome que identificará a campanha criada pelo usuário, como por exemplo: **First campain**, o usuário deverá continuar a seguir os passos para configurar a campanha.
Conforme mostrado acima, há quatros campos os quais são destinados à inserção de **Contatos**, **Mensagens**, **Horários** e **Configurações** que devem ser preeenchidos conforme as necessidades do próprio usuário.

<Frame>
  <img
    className='block dark:hidden'
    src='images/campain/add-ticket-light.png'
    alt='Add ticket light'
  />
  <img
    className='hidden dark:block'
    src='images/campain/add-ticket-dark.png'
    alt='Add ticket dark'
  />
</Frame>

Ao navegar no menu o usuário encontrará um campo para adicionar suas etiquetas, toque no botão **Adicionar etiquetas** para inserir a etiqueta desejada. Estas etiquetas podem ser criadas para identificar detalhes da campanha, promoções e muito mais.
Ao tocar no botão uma nova janela será aberta onde o usuário podera filtar etiquetas existentes para adicionar, esse campo permite a insersão de etiquetas já existentes onde mais de uma etiqueta poderá ser utilizada para uso na campanha.

<Note>
  Caso o usuário deseje excluir uma etiqueta adicionada basta tocar no icone de
  lixo na etiqueta.
</Note>
