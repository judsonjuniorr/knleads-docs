title: Criando uma nova campanha

---

<Note>
  Conforme mostrado na sessão anterior o usuário deverá seguir os passos do menu
  **+ Nova campanha** e inserir um nome para a campanha. Esse nome pode ser
  personalizado usando letras, números e caracteres especiais.
</Note>

## Visualizando o menu de nova campanha

Ao inseir o nome da campanha o usuário deverá pressionar o botão **Continuar** para ser direcionado para o menu onde poderá montar um campanha. Realizando esta ação o usuário visualizará os campos confome o modelo.

No topo do menu será mostrado o nome que identificará a campanha criada pelo usuário, como por exemplo: **First campain**. Existem quatros campos os quais são destinados à inserção de **Contatos**, **Mensagens**, **Horários** e **Configurações** que devem ser preenchidos conforme as necessidades do próprio usuário.

## Adicionando e removendo etiquetas

Ao navegar no menu o usuário encontrará um campo para adicionar suas etiquetas, toque no botão **Adicionar etiqueta** para inserir a etiqueta desejada. Estas etiquetas podem ser criadas para identificar detalhes da campanha como lançamentos, promoções, orçamentos e muito mais.

Ao tocar no botão uma nova janela será aberta onde o usuário podera filtar etiquetas para adicionar, esse campo permite a insersão de etiquetas já existentes onde mais de uma etiqueta poderá ser utilizada para identificar a campanha. Após adicionar uma etiqueta ela será listada conforme o modelo.

<Note>
  Caso o usuário deseje excluir uma etiqueta adicionada basta tocar no botão
  **Remover** na etiqueta.
</Note>

## Adicionando e removendo contatos

Continuando a navegação no menu, o usuário encontrará um campo para adicionar seus contatos, toque no botão **Adicionar contato** para inserir um ou mais contatos desejados. Os contatos inseridos na campanha serão o alvo dos disparos gerados pela campanha durante o período de andamento da campanha.
