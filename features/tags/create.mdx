---
title: Criando uma etiqueta
description: 'Aprenda a criar uma etiqueta no sistema'
---

Após acessar a seção de etiquetas, clique no botão **Nova Etiqueta** na parte superior direita.

Uma modal será aberta para que você possa preencher com o nome da etiqueta.

  <img

  />
</Frame>

Após preencher o nome da etiqueta, clique no botão **Cadastrar**.

<Warning>**Atenção:** Não é possível criar duas etiquetas com o mesmo nome dentro de um espaço de trabalho.</Warning>
