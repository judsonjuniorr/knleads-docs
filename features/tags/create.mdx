---
title: Criando uma tag
description: 'Aprenda a criar uma tag no sistema'
---

Após acessar a seção de tags, clique no botão **Nova TAG** na parte superior direita.

Uma modal será aberta para que você possa preencher com o nome da tag.
<Frame>

  <img
    className="hidden dark:block"

    alt="Create Tag Dark"
  />
</Frame>

Após preencher o nome da tag, clique no botão **Cadastrar**.

<Warning>**Atenção:** Não é possível criar duas tags com o mesmo nome dentro de um espaço de trabalho.</Warning>
