---
title: Excluindo uma tag
description: 'Aprenda a remover uma tag no sistema'
---

Após acessar a seção de tags, localize a tag que deseja excluir e clique no botão com 3 pontos e em seguida em **Deletar** conforme a imagem abaixo.

<Frame>
  <img
    className="block dark:hidden"
    src="/images/tags/tag-delete-light.png"
    alt="Delete Tag Light"
  />
  <img
    className="hidden dark:block"
    src="/images/tags/tag-delete-dark.png"
    alt="Delete Tag Dark"
  />
</Frame>

Uma janela de confirmação será aberta para evitar que a tag seja excluída acidentalmente.

<Frame>
  <img
    className="block dark:hidden"
    src="/images/tags/tag-delete-modal-light.png"
    alt="Delete Tag Modal Light"
  />
  <img
    className="hidden dark:block"
    src="/images/tags/tag-delete-modal-dark.png"
    alt="Delete Tag Modal Dark"
  />
</Frame>

Preencha o nome da tag e clique no botão **Deletar**.

<Note>A exclusão de uma tag não afeta os contatos que já foram marcados com ela.</Note>
<Warning>**Atenção:** A exclusão de uma tag é irreversível.</Warning>
