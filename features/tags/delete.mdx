---
title: Excluindo uma etiqueta
description: 'Aprenda a remover uma etiqueta no sistema'
---

Após acessar a seção de etiquetas, localize a etiqueta que deseja excluir e clique no botão com 3 pontos e em seguida em **Deletar** conforme a imagem abaixo.

<Frame>
  <img
    className="block dark:hidden"
    src="/images/tags/tag-delete-light.png"
    alt="Delete label Light"
  />
  <img
    className="hidden dark:block"
    src="/images/tags/tag-delete-dark.png"
    alt="Delete label Dark"
  />
</Frame>

Uma janela de confirmação será aberta para evitar que a etiqueta seja excluída acidentalmente.

<Frame>
  <img
    className="block dark:hidden"
    src="/images/tags/tag-delete-modal-light.png"
    alt="Delete label Modal Light"
  />
  <img
    className="hidden dark:block"
    src="/images/tags/tag-delete-modal-dark.png"
    alt="Delete label Modal Dark"
  />
</Frame>

Preencha o nome da etiqueta e clique no botão **Deletar**.

<Note>A exclusão de uma etiqueta não afeta os contatos que já foram marcados com ela.</Note>
<Warning>**Atenção:** A exclusão de uma etiqueta é irreversível.</Warning>
