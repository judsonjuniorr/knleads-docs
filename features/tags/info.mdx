---
title: O que são etiquetas?
description: 'Conheça as etiquetas e como elas podem te ajudar a organizar seus contatos.'
---

As etiquetas são uma forma de organizar seus contatos, elas podem ser utilizadas para separar seus contatos por grupos, por exemplo: clientes, fornecedores, parceiros, etc.

<Note>As etiquetas são exclusivas para cada espaço de trabalho, ou seja, se você possui mais de um espaço de trabalho, as etiquetas de um espaço de trabalho não serão exibidas em outro.</Note>

Na criação de campanha é possível utilizar as etiquetas para segmentar os contatos que receberão a campanha, por exemplo: enviar uma campanha apenas para os contatos que possuem a etiqueta "clientes".
