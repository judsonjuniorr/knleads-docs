---
title: O que são tags?
description: 'Conheça as tags e como elas podem te ajudar a organizar seus contatos.'
---

As tags são uma forma de organizar seus contatos, elas podem ser utilizadas para separar seus contatos por grupos, por exemplo: clientes, fornecedores, parceiros, etc.

<Note>As tags são exclusivas para cada espaço de trabalho, ou seja, se você possui mais de um espaço de trabalho, as tags de um espaço de trabalho não serão exibidas em outro.</Note>

Na criação de campanha é possível utilizar as tags para segmentar os contatos que receberão a campanha, por exemplo: enviar uma campanha apenas para os contatos que possuem a tag "clientes".
