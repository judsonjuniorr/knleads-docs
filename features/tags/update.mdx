---
title: Atualizando uma tag
description: 'Aprenda a alterar uma tag no sistema'
---

Após acessar a seção de tags, localize a tag que deseja alterar e clique no botão com 3 pontos e em seguida em **Editar** conforme a imagem abaixo.

<Frame>
  <img
    className="block dark:hidden"
    src="/images/tags/tag-edit-light.png"
    alt="Edit Tag Light"
  />
  <img
    className="hidden dark:block"
    src="/images/tags/tag-edit-dark.png"
    alt="Edit Tag Dark"
  />
</Frame>

Uma modal será aberta para que você possa preencher com o novo nome da tag.

Após preencher o nome da tag, clique no botão **Editar**.

<Warning>**Atenção:** Não é possível ter duas tags com o mesmo nome dentro de um espaço de trabalho.</Warning>
