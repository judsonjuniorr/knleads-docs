---
title: Atualizando uma etiqueta
description: 'Aprenda a alterar uma etiqueta no sistema'
---

Após acessar a seção de etiquetas, localize a etiqueta que deseja alterar e clique no botão com 3 pontos e em seguida em **Editar** conforme a imagem abaixo.

<Frame>
  <img
    className="block dark:hidden"
    src="/images/tags/tag-edit-light.png"
    alt="Edit label Light"
  />
  <img
    className="hidden dark:block"
    src="/images/tags/tag-edit-dark.png"
    alt="Edit label Dark"
  />
</Frame>

Uma modal será aberta para que você possa preencher com o novo nome da etiqueta.

Após preencher o nome da etiqueta, clique no botão **Editar**.

<Warning>**Atenção:** Não é possível ter duas etiquetas com o mesmo nome dentro de um espaço de trabalho.</Warning>
