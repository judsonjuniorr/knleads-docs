Precisaremos ter as seguintes informações:

<ParamField path="Email">
  Seu email é o nome de usuário que você utiliza para acessar o sistema.
</ParamField>
<ParamField path="Senha">
  A senha é o código de segurança que você utiliza para acessar o sistema.
</ParamField>

<Note>
  Vale lembrar que para segurança o sistema exige que a senha tenha mais de 8 dígitos contendo números, letras e caracteres especiais.
</Note>
